## English：

Personal repository for network proxy configuration files and rulesets.  
(Special thanks to Ms. Cecilia for providing the VPS service used for testing.)

## 📁 Directory Structure

* **Stash**: Contains custom override (`.stoverride`) configuration files for the Stash app.
  * `Stash_Override_CN.stoverride`: Custom override configuration file specialized and optimized for the network environment in China.
  * `Stash_Override_RU.stoverride`: Custom override configuration file specialized and optimized for the network environment in Russia.
  * `Stash_Override_IR.stoverride`: Custom override configuration file specialized and optimized for the network environment in Iran.
## Private customized files
  * `Stash_Override_No1.stoverride`: Primary override configuration file, optimized for daily use.
  * `Stash_Override_No2.stoverride`: Secondary override configuration file, designated for poor network conditions.
  * `Stash_Override_Test.stoverride`: Testing configuration file, utilized for ongoing iterations, rule debugging, and network testing.
  * `config.stoverride`: General backup for the baseline override configuration.

## ⚠️ Terms of Use

* For remote links, please use the Raw direct URL generated by GitHub containing the `/refs/heads/main/` path. Paste it directly into the remote override section within Stash.
* This repository and its configurations are intended solely for personal technical research, network optimization, and privacy protection.
* Major remote resources referenced in the configuration files and acknowledgments:
  * **Rulesets**:
    * `BanAD` / `BanProgramAD` rulesets (Sourced from the open-source project [ACL4SSR/ACL4SSR](https://github.com/ACL4SSR/ACL4SSR))
    * `AdvertisingLite` / `ChinaMax` / `Privacy` rulesets (Sourced from the open-source project [blackmatrix7/ios_rule_script](https://github.com/blackmatrix7/ios_rule_script))
  * **GEOIP Databases**: (Automatically updated via Stash's built-in mechanism)
